Linda UI is a React 19 design system built on top of HeroUI v3 and react-aria-components. It ships three focused packages — brand tokens, ready-to-use components, and full-page blocks — so you can go from `pnpm add` to a polished, accessible UI in minutes.

## Get started in three steps

<Steps>
  <Step title="Install the packages">
    Add all three Linda UI packages and their peer dependencies to your project.

    ```bash
    pnpm add @lindaui/ui @lindaui/blocks @lindaui/tokens
    pnpm add react@^19 react-dom@^19 react-hook-form@^7
    ```
  </Step>
  <Step title="Import the token bundle">
    Pull in the OKLCH teal tokens, HeroUI defaults, and Tailwind v4 utilities by importing the CSS bundle once — typically in your root layout or global stylesheet.

    ```ts
    import "@lindaui/tokens/css";
    ```
  </Step>
  <Step title="Use a component">
    Import any component from `@lindaui/ui` or a pre-built section from `@lindaui/blocks` and render it straight away — no extra configuration required.

    ```tsx
    import { Button } from "@lindaui/ui";

    export default function App() {
      return <Button variant="primary">Get started</Button>;
    }
    ```
  </Step>
</Steps>
